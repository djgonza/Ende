Proyecto: zuul-malo 
Autores: Michael Kolling y David J. Barnes 

Este proyecto forma parte del material para el libro 

    En primer lugar los objetos con Java 
    - Una introducción práctica usando BlueJ 
    Tercera edición 
    David J. Barnes y Michael Kolling 
    Pearson Education, 2006 

Este proyecto es un simple marco para un juego de aventuras. En esta versión, 
tiene unas pocas habitaciones y la capacidad de un jugador de ir a pie entre
estas habitaciones. Eso es todo. 

Para iniciar esta aplicación, crear una instancia de la clase "juego" 
y llame a su mëtodo "jugar". 

Esta versión del juego contiene algunas decisiones de diseño muy malas. No debe 
ser utilizado como una base para ampliar el proyecto sin la correccion de estos
problemas de diseño. Sirve como un ejemplo para discutir lo bueno y lo malo del
diseño (capítulo 7 del libro). 

El capítulo 7 del libro contiene una descripción detallada de los problemas en este 
proyecto, y cómo solucionarlos. 

El proyecto 'zuul-mejorado' contiene una versión de este proyecto con una mejor 
diseñado estructura de clases. Se incluyen las correcciones discutidas en el libro.
